Dog Breed Classifier

This is a project I made using PyTorch for my Udacity Deep Learning program. It is an image classifier that will determine if an image is of a person or a dog, and will return the closest breed of dog that the input image resembles.

The VGG16 convolutional neural network is used to determine if a dog is present in the image. OpenCV is used for face detection to determine if a person is present, in order to show the appropriate message. If the photo is of a dog, the breed is determined from a network I made with transfer learning from the VGG16 network.
