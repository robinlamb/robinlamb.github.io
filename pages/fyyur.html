Fyyur Web Development Project

This is a project from my Udacity Full Stack Web Developer program. In this project, I set up a Postgres database to interact with the given Flask web application and used SQLAlchemy to interact with the database. I used SQLAlchemy to define relationships between the tables, and write queries to get information from the database. I also set up Flask Migrations for the database, which store versions of the database in order to allow changes to be rolled back easily if there is a problem with the changes.
