Paper Rock Scissors Android App

This is a Paper, Rock, Scissors game that I made for Android that can be played by one or two players using the same device. A separate Java class figures the game logic given the two choices. If the game is in one player mode, a random choice is generated by the computer opponent.

I made XML layouts and included different layouts for the SelectionActivity and OutcomeActivity in different screen orientations to make use of the whole screen in either portrait or landscape orientation. I included XML values for smallest width of screens 600dp and 800dp to make the text size, spacing, etc. scale up to look better on tablets. I included different sizes of the images in the app to support different screen resolutions.

I used animations to make the text fade in and appear to flash, and to make the pictures appear to bounce when selected. I included sounds to play for the different outcomes. I included a shared preference to save the user's preference to have the audio on/off.
